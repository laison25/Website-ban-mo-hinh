# Website Bán Mô Hình

## Thông tin nhóm
- **Lại Sơn** – Nhóm trưởng
- **Phương** – Thành viên
- **Vinh** – Thành viên

## Giới thiệu
Đây là dự án xây dựng website bán mô hình nhằm hỗ trợ việc giới thiệu, trưng bày và bán các sản phẩm mô hình trên nền tảng web.  
Website hướng đến giao diện dễ nhìn, dễ sử dụng và có các chức năng cơ bản phục vụ cả người dùng và quản trị viên.

## Mục tiêu dự án
- Xây dựng một website bán mô hình đơn giản, dễ sử dụng
- Hiển thị danh sách các sản phẩm mô hình
- Cho phép người dùng xem thông tin chi tiết của từng sản phẩm
- Hỗ trợ tìm kiếm sản phẩm theo tên hoặc danh mục
- Xây dựng chức năng giỏ hàng và đặt hàng cơ bản
- Xây dựng trang quản lý để admin có thể quản lý sản phẩm

## Nội dung đã thực hiện
Hiện tại nhóm đã thực hiện được các nội dung sau:
- Tạo repo GitHub cho dự án
- Xây dựng nội dung README ban đầu
- Viết tài liệu SRS cho chức năng đăng nhập hệ thống
- Thiết kế giao diện mẫu trên Figma cho các trang chính:
  - Trang đăng nhập
  - Trang chủ
  - Trang chi tiết sản phẩm
- Xây dựng website bằng PHP
- Kết nối cơ sở dữ liệu MySQL
- Import dữ liệu sản phẩm vào database
- Hiển thị danh sách sản phẩm từ database
- Xây dựng trang chi tiết sản phẩm
- Xây dựng chức năng đăng nhập và đăng ký tài khoản
- Xây dựng giỏ hàng và trang thanh toán
- Xây dựng trang lịch sử đơn hàng của người dùng
- Xây dựng trang admin quản lý sản phẩm
- Hỗ trợ upload ảnh sản phẩm từ trang admin

## Nội dung đang tiếp tục hoàn thiện
- Chỉnh sửa giao diện website bám sát mẫu Figma hơn
- Tối ưu hiển thị ảnh sản phẩm
- Hoàn thiện thêm các tài liệu SRS cho các chức năng khác
- Hoàn thiện chức năng quản lý đơn hàng
- Kiểm tra và sửa lỗi hệ thống

## Đối tượng sử dụng
- **Người dùng/khách hàng**: xem sản phẩm, tìm kiếm, thêm vào giỏ hàng, đặt hàng và xem lịch sử đơn hàng
- **Quản trị viên**: quản lý sản phẩm, hình ảnh sản phẩm và theo dõi đơn hàng

## Phân công sơ bộ
- **Lại Sơn**: phụ trách README, tài liệu SRS, thiết kế giao diện Figma và triển khai website bằng PHP/MySQL
- **Phương**: hỗ trợ định hướng nội dung dự án và phối hợp xây dựng ý tưởng giao diện
- **Vinh**: phụ trách phân tích chức năng giỏ hàng, đặt hàng và hỗ trợ hoàn thiện hệ thống

## Công nghệ sử dụng
- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Công cụ thiết kế giao diện**: Figma
- **Quản lý mã nguồn**: GitHub
- **Môi trường chạy thử**: XAMPP
