# Website bán mô hình PHP

Project demo giao diện PHP theo mẫu Figma mà bạn gửi, gồm:
- Trang chủ: index.php
- Trang đăng nhập: login.php
- Trang chi tiết sản phẩm: product-detail.php?id=1

## Cách chạy bằng XAMPP
1. Giải nén folder này vào htdocs
2. Mở XAMPP và bật Apache
3. Truy cập:
   - http://localhost/website-ban-mo-hinh-php/
   - hoặc http://localhost/website-ban-mo-hinh-php/index.php

## Ghi chú
- Project này dùng PHP + HTML + CSS + JS
- Dữ liệu sản phẩm đang là dữ liệu mẫu trong data/products.php
- Ảnh đang là SVG placeholder để bạn thay bằng ảnh thật xuất từ Figma hoặc ảnh sản phẩm thật
- Khi cần, có thể nối thêm MySQL để làm đăng nhập, giỏ hàng, đặt hàng thật
